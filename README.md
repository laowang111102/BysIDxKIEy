# 基于Python Django的个人博客系统 python640

## 项目概述

本项目是一个基于Python Django框架的个人博客系统，具备用户登录注册、博客数据展示、博客评论、博客发布修改删除等基本功能。系统后端使用了Django自带的admin管理后台，实现了对各个数据字段的增删改查操作。

## 技术栈

- Python
- Django
- HTML/CSS/JavaScript
- MySQL（数据库）

## 功能模块

### 基础功能

- 用户登录注册
- 博客数据展示
- 博客评论
- 博客发布、修改、删除
- 登录使用滑动验证
- 富文本框

### 管理后台

- 使用django-admin构建超级管理员管理后台
- 对各个数据字段的增删改查功能

## 系统角色

- 普通用户：可进行登录注册、浏览博客、评论博客、发布修改删除自己的博客
- 超级管理员：具备系统管理权限，可管理所有用户数据及博客内容

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 配置MySQL数据库
4. 运行项目
